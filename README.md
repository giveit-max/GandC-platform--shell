# G&C Platform Shell (Vercel Ready)

This is the starter shell for the G&C platform, including:

- Next.js 15 App Router
- Login page
- Founder dashboard page
- Vercel deployment config
- Basic folder structure

## Folders

- /app — main app pages
- /app/login — founder login
- /app/founder — dashboard shell
- /components — shared UI components
- /styles — global styles

## Deployment

This project includes `vercel.json` and is ready for Vercel deployment.
